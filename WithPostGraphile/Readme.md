# Postgraphile playground

## Setup dev env

1. Copy `app/template.env` to `app/.env` and modify as needed.

2. Copy `db/template.env` to `db/.env` and modify as needed.

3. Copy `db/pass_file.template` to `db/pass_file` and modify as needed.

4. Create directory `./db/db-data` if it doesn't exist yet.
